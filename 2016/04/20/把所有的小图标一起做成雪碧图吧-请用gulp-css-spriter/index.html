用gulp-css-spriter很简单。

第一步: 在某个文件夹用shitf+鼠标右键

第二步: npm install gulp-css-spriter

https://www.npmjs.com/package/gulp-css-spriter (官网gulp插件)

第三步:在gulpfile.js 文件

第四步：打开生成的dist/css
